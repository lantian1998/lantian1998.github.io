加入 DN42 实验网络

2020-03-16 提示 ¶本文已有更新版本：参见《DN42 实验网络介绍（2020 版）》。新版介绍中有更详细的注册申请流程，并根据 DN42 三年来的变化做出了修改。以下内容写于 2017 年，仅作存档用途。DN42 全称 Decentralized Network 42（42 号去中心网络），是一个大型的 VPN 网络。但是与其它传统 VPN 不同的是，DN42 使用了大量在互联网骨干上应用的技术（例如 BGP），可以很好的模拟一个真实的网络环境。正因为它的真实，使用 DN42 的门槛比较高。你要扮演一个 ISP（互联网服务提供商），注册一个 ASN 号码，注册 IPv4 和 IPv6 的地址池，并且使用 BGP 在自己的服务器上广播它们。你还要和其它的用户联系，和他们做 Peering（对接），一步步进入完整的 DN42 网络。DN42 在 172.20.0.0/14 和 fd00::/8 上运行，而这两个 IP 段都是分配给内网使用的。换句话说，你在 DN42 上怎么折腾，都不会影响到服务器其它的互联网连接。DN42 官方提供了很详细的注册 ASN 和 IP 的教程，只需一步一步照着做就可以，因此本文不会涉及该方面。本文的重点在地址池的广播和与其它用户的 Peering 上面。本文假定读者已经完成了 ASN、地址池的注册和设置，如图所示：首先，...
